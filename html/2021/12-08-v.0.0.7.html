Версия 0.0.7

Выпущено обновление транслятора, библиотеки и сборщика Android приложений.

  • Исправлена статическая и динамическая диагностика неинициализированных переменных в VAR-аргументах предопределённых процедур
  • Устранена избыточная пометка переменных в вызовах предопределённых процедур как используемых в качестве VAR-аргументов, что приводило к ухудшению производительности сгенерированного в Java и JavaScript кода
  • Воплощена генерация кода FOR в Java и JavaScript в случае, когда итератор также используется как VAR-аргумент
  • Устранена избыточная генерация в Оберон модулей из сингулярности
  • Исправление вызова команды version
  • Поправлена генерация в Активный Оберон, более точно учтены особенности численных типов
  • Улучшено форматирование генерируемого кода на Java

Расширена настраиваемость вызова сборщика apk-файлов. Потребность, в основном, вызвана добавленной возможностью собирать Android-приложения непосредственно на самом Android.

В библиотеке по умолчанию добавлена возможность рисования текста и обработки касаний в Android, исправлено несколько ошибок, изменена суть Chars0X.CopyChar, и добавлено несколько модулей для печати и заполнения массивов.
